Collapsible snippets

 Posted on 10 April 2021

Just set a front matter value, and voilà!

hello.hpp 
#pragma once

void hello_snippets();
hello.cpp 
#include "world.hpp"

#include <iostream>

void hello_snippets() {
    std::cout << "Hello snippets!\n";
}

This is another set of snippets:

world.hpp 
#pragma once

void hello_world();
world.cpp 
#include "world.hpp"

#include <iostream>

void hello_world() {
    std::cout << "Hello world!\n";
}
