Code snippets

 Posted on 9 April 2021

It’s pretty simple, just use the corresponding include:

hello.hpp
#pragma once

void hello_snippets();
hello.cpp
#include "world.hpp"

#include <iostream>

void hello_snippets() {
    std::cout << "Hello snippets!\n";
}

This is another set of snippets:

world.hpp
#pragma once

void hello_world();
world.cpp
#include "world.hpp"

#include <iostream>

void hello_world() {
    std::cout << "Hello world!\n";
}

You can insert a code snippet using the regular Markdown syntax without any highlighting:

#!/usr/bin/env bash

set -o errexit -o nounset -o pipefail

main() {
    echo "Hello, world!"
}

main
